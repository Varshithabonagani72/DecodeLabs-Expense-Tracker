# DecodeLabs Expense Tracker

## Project Overview

• This project is a simple Expense Tracker application developed using Python during my internship at DecodeLabs

• The application helps users enter and calculate daily expenses

• This project is based on mathematical operations and accumulator logic in Python

## Features

• Add expense amounts

• Calculate total expenses

• Continuous data entry

• Simple and user-friendly command line interface

## Technologies Used

• Python

## Project File

• expense_tracker.py – Main Python program

## How to Run the Project

• Install Python on your system

• Open Command Prompt or PowerShell

• Navigate to the project folder

• Run the command:

```bash id="x1d8rp"
python expense_tracker.py
```
